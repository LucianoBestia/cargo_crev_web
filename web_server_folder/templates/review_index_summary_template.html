cargo-crev

cargo-crev is a cryptographically verifiable code review system for the cargo (Rust) package manager:
https://github.com/crev-dev/cargo-crev

cargo-crev runs locally on the developer computer.
It needs to be installed and initialized in a few steps.
It takes some time to initialize it and learn how to use it. cargo-crev is a relatively new project and the majority of rust developers don't have it installed yet.

cargo_crev_web

I created cargo_crev_web to enable all developers to view crev reviews.
There is no installation, initialization or learning needed. It is just a simple web page.
It simply shows a list of reviews for a crate.
I wish that crates.io will have some day a link to cargo_crev_web just like today it has links to documentation and repository.

web of trust

New crev reviews are written by developers and committed to crev review repos and pushed to GitHub. From there they are available to all.
We need a system to trust this persons to trust their reviews. cargo-crev has a list of trusted persons. But we personally only know a limited number of trusted persons. To make this circle bigger we need more people to trust. When we trust somebody for reviews, we usually also trust their choices to trust other people. So we can trust also them. A little less, but still trust them.
This is an important part of the web of trust.
It is common in every day life that the friends of our friends are probably also our friends.

cargo crev reviews and advisory

It is recommended to always use
https://github.com/crev-dev/cargo-crev
to verify the trustworthiness of each of your dependencies.
Please, spread this info.
